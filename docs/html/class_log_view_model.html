Detailed Description

Current logging model viewable from UI directly using DataContext.

Represents the current logging model, viewable directly from the UI via data binding.

This class provides an ObservableCollection<T> of log messages that can be displayed in real-time within the UI. The Logs collection is updated on the UI thread to ensure thread safety.
